# Walk-Up Songs

A mobile-friendly PWA for managing softball team walk-up songs with Spotify integration.

## Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `walkup-songs`)
2. Upload all files to the repository root
3. Go to **Settings → Pages → Branch: main → Save**
4. Your site will be live at `https://yourusername.github.io/walkup-songs/`

## Add to iPhone Home Screen

1. Open the site in Safari
2. Tap the **Share** button (box with arrow)
3. Tap **Add to Home Screen**
4. Tap **Add**

The app will open full-screen like a native app.

## Generating App Icons (optional, for better iOS icon)

1. Open `generate-icons.html` in a browser
2. Click **Download Both Icons**
3. Save `icon-192.png` and `icon-512.png` to the project folder
4. Delete `generate-icons.html` before uploading to GitHub

## Features

- **Roster tab** — Add players with jersey number, name, song name, and Spotify link
- **Game tab** — Cycle through the batting order with Spotify player embedded
- All data saved locally in the browser (no account needed)
- Works offline after first load
